# Virtual-Pokedex

A Java-based desktop Pokedex application designed to help trainers manage their Pokémon collection and analyze team performance.

## Features
* **Search & Discover:** Real-time search functionality with auto-complete suggestions using a Trie data structure.
* **Team Management:** Create teams, add your favorite Pokémon, and view/remove members from your roster.
* **Versus Arena:** Compare two Pokémon side-by-side with a detailed stat breakdown (HP, Attack, Defense, Speed, and BST).
* **Team Analytics:** Analyze your team's overall power, average stats, and type distribution.
* **Interactive UI:** Modern Java Swing interface with dynamic backgrounds based on Pokémon types.

## Tech Stack
* **Language:** Java
* **GUI:** Java Swing
* **Data Structure:** Trie (for search suggestions)
* **Database:** MySQL (via JDBC)
* **Build Tool:** Maven
* **Containerization:** Docker
